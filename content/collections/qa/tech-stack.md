Tôi làm việc chủ yếu với JavaScript/TypeScript ecosystem.

**Frontend**: React, Next.js (App Router), Tailwind CSS. Tôi quan tâm nhiều đến performance và UX — Server Components, code splitting, và animation có chủ đích.

**Mobile**: React Native + Expo. Cross-platform nhưng native feel.

**Backend**: Node.js với Express hoặc Fastify. Với database: PostgreSQL cho relational data, Redis cho caching và real-time features, Firebase khi cần quick prototyping.

**Tools**: TypeScript everywhere, Git, Docker cho local dev, Vercel/Railway cho deployment.

Tôi chọn tools dựa trên bài toán — không phải hype. Và tôi đủ linh hoạt để học thêm khi dự án cần.
